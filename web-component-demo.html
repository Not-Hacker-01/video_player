üîß Available Attributes

Configure your video player using HTML attributes:

<video-ad-player-sdk videoid="HfUGeWXS-bE" // YouTube video ID position="bottom right" // Position: top/bottom + left/right/center size="medium" // Size: small, medium, large, custom width="320" // Custom width (when size="custom") height="180" // Custom height (when size="custom") muted="true" // Mute the video closeable="true" // Show close button zindex="9999" // CSS z-index > </video-ad-player-sdk>

üìù Multiple Video Sources

üéØ Event Handling

Listen to player events using standard DOM events:

<script> const player = document.querySelector('video-ad-player-sdk'); player.addEventListener('ready', (event) => { console.log('Player is ready:', event.detail.player); }); player.addEventListener('close', () => { console.log('Player was closed'); }); player.addEventListener('error', (event) => { console.error('Player error:', event.detail); }); </script>
